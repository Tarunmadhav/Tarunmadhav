# 🚀 Setup — Deploy Tarun's Animated GitHub Profile

## Step 1 — Create the magic repo

GitHub turns a repo named **exactly your username** into your profile page.

```bash
# On GitHub.com → New repository → name it exactly "Tarunmadhav" → Public
# Then clone it locally:
git clone https://github.com/Tarunmadhav/Tarunmadhav.git
cd Tarunmadhav
```

## Step 2 — Copy all files from this package into the repo

```
Tarunmadhav/
├── README.md                          ← terminal-style profile layout
├── tarun-ascii.svg                    ← your ASCII portrait (pre-generated) ✅
├── info-card.svg                      ← neofetch card (pre-generated) ✅
├── contrib-heatmap.svg                ← contribution heatmap (pre-generated) ✅
├── source-prepped.png                 ← your prepped photo (keep for regenerating portrait)
├── data/
│   └── contributions.json             ← generated by workflow, commit the one here ✅
└── scripts/
    ├── requirements.txt
    ├── fetch_contributions.py         ← scrapes GitHub public HTML, no token needed
    ├── render_heatmap_svg.py          ← renders animated SVG from JSON
    ├── make_ascii_svg.py              ← converts prepped photo to typing SVG
    ├── make_info_card.py              ← generates the neofetch info card
    └── prep_photo.py                  ← background removal + CLAHE (run locally only)
```

## Step 3 — Push everything and enable Actions

```bash
# Copy all the files in, then:
git add .
git commit -m "feat: animated terminal profile"
git push origin main
```

Go to **Settings → Actions → General** → set **Workflow permissions** to  
**"Read and write permissions"** → Save.

## Step 4 — Trigger the first heatmap refresh

Go to **Actions tab** → click **"Update profile art"** → **Run workflow**.

It will:
1. Scrape your real contribution data from `github.com/users/Tarunmadhav/contributions`
2. Render a fresh animated `contrib-heatmap.svg`
3. Commit it back automatically

After ~30 seconds your profile is live with real data ✅

The workflow re-runs daily at 11:47 AM IST automatically.

## Updating your photo (optional, do locally)

When you want a new portrait:

```bash
# Install heavy local deps (only once)
pip install pillow numpy opencv-python rembg

# 1. Remove bg + boost contrast
python scripts/prep_photo.py your-new-photo.jpg

# 2. Generate the typing SVG
python scripts/make_ascii_svg.py

# 3. Commit and push
git add source-prepped.png tarun-ascii.svg
git commit -m "chore: update portrait"
git push
```

## Updating the neofetch card

Edit `ROWS` in `scripts/make_info_card.py`, then:

```bash
python scripts/make_info_card.py
git add info-card.svg
git commit -m "chore: update info card"
git push
```

## How it all works

| File | What it does | Needs updating |
|------|-------------|----------------|
| `tarun-ascii.svg` | Your face in ASCII — types itself in with SMIL animation | Only when photo changes |
| `info-card.svg` | Neofetch panel — fades in line by line | When job/stack changes |
| `contrib-heatmap.svg` | Real 53-week contribution calendar — boxes animate in diagonally | **Daily, auto** |

No third-party stats services, no GitHub token, no JavaScript. Everything is  
self-contained SVG committed to your own repo.
